Directory Lister
================

> Browse files of the directory.


Usage
-----

Syntax for the content tag:

```html
<rs-module>
{
	"DirectoryLister":
	{
		"directory": "directory",
		"options":
		{
			"hash": "directory",
			"replace":
			{
				".": "",
				"{extension}": ""
			}
		}
	}
}
</rs-module>
```
